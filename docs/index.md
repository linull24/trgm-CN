# TRGM-CN

Terr***ram 是截止到2024年的一个非常有影响力的英文传播出版物。但烦请读者先容许这冗长的前言与译者的话。

其意义在于`MPI-Broadcast`。

在译者之前的对于2083的微薄工作之中，意识到了对于形态组织上的观点而言，安德斯·贝林·布雷维克**很早就已经达到了远比译者熟的程度——意识到了在组织上，组织形态的技术性无法对抗已经占据优势的一方**，于是主动降低对于组织的目标的追求，乃至于组织本身都是一个阶段性的产物。

广播，回音，是否胆敢从成为那灯塔？

Terr***ram 的英雄作者们已经为此付出了代价，而中文世界对于这种广播本身的了解都不熟悉。读这本书的是谁我都不在乎，CN must be synchronized.

不要期待广播之后的反馈，但是要认真广播

---

另外，由于技术的限制和对于文本的安全性考量，这系列译本全部是在 material-mkdocs 上进行部署。但是，原本的**美学**设计非常值得分析。非常建议读者应该主动下载一个版本。
